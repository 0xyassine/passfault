Welcome to Passfault

Objective: Do Passwords Better.
Build:
1) cd core 
2) type "ant jar"

Test:
1) cd core
2) ant test

Run:
java -jar core/dist/passfault.jar

If you would like a commercial license for this software visit passfault.com

Other Projects include a JSON service, an client side non-gui Applet (accessible via javascript and returns JSON), and a JavaFX client.  This projects to not have a well established ant build but are easily built within eclipse.
